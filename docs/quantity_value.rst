.. _quantity_value:

**************
Quantity value
**************

.. contents::
   :local:

'Quantity value' is a term for a measured value paired with units of measurement. ``QV`` supports the notion of quantity values though several functions defined in the :mod:`.quantity_value` module. 

The function :func:`.qvalue` creates a quantity-value, :func:`.qratio` creates a quantity-value that is a dimensionless ratio and :func:`.qresult` resolves the units of an expression involving quantity-values.

More information is given in the :ref:`examples` section.
