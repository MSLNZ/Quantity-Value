******
Prefix
******

.. contents::
   :local:

The :mod:`.prefix` module defines standard metric and binary prefixes: ``yocto`` through to ``yotta`` and ``kibi`` through to ``yobi``. There is also a function to generate a set of SI mass units (as this is a special case, with the kilogram being the conventional reference unit).
